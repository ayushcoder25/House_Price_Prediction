# House_Price_Prediction

## About Project

This project predicts house prices using machine learning.
User can enter details like area, bedrooms, bathrooms etc. and the model gives estimated price.

I made this project to understand how ML works on real data.

---

## What it does

* Takes input from user
* Uses trained model
* Predicts house price

---

## Features used

* area
* bedrooms
* bathrooms
* stories
* parking
* air conditioning
* main road

---

## Tech used

* Python
* Pandas
* Scikit-learn
* Streamlit

---

## How to run

Install libraries:

```id="l7w8ny"
pip install pandas scikit-learn streamlit
```

Run app:

```id="8n3zqk"
streamlit run model/app.py
```

---

## Project files

* train_model.py → used to train model
* app.py → user interface
* Housing.csv → dataset
* model.pkl → saved model
